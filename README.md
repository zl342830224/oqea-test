# oqea-test-application

A Vue App for a small test which can CRUD medicine supports

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/)

### How to Run it

Now, enter following commands to run it :)
----bash
cd oeqa-test

npm install

npm run jsondb

//open other terminal
npm run dev

---
